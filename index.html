bannerMobile

1. Установка

Подключаем скрипт перед закрывающимся тегом body

  1. <!-- Compiled and minified JavaScript -->
  2. <script src = "https://widget-lib.vercel.app/bannerMobile-build/main.js">
  3. </ script>

2. Запуск
  1. topdent.bannerMobile ( { } )

Параметры Значение Коментарий
link string Вставялем прямую ссылку
time number Время смены слайдов
top number Размер прокрутки сверху
text array Текст и картинка

Пример:

  6.             text: [{
  7.                     name: 'Элайнеры  <br / > По цене брекетов',
  8.                     image: '/img-1.jpg',
  9.                 },

youtubeWidget

1. Установка

Подключаем скрипт перед закрывающимся тегом body

  1. <!-- Compiled and minified JavaScript -->
  2. <script src = "https://widget-lib.vercel.app/youtubeWidget-build/main.js">
  3. </ script>

2. Запуск
  1. topdent.widgetVideo ( { } )
Параметры Значение Коментарий
videoId string Берем из ссылки на youtube , пример HF1dGDQO6Kc
position strung left and right
marginSlide number отступ по сторонам
marginBottom number отступ c с верху
orientation string vertical, по умолчанию горизонтальный
